## About The Project
This tool is a professional **IP and Port Scanner** developed by **X_Shuvo_X**. It is designed for security testing and network analysis. With a clean and colorful terminal interface, it helps you identify the IP address of any domain and check the status of specific ports.

- **Developer:** X_Shuvo_X
- **Organization:** Bhaluka Cyber Security

---

## Features
- **Link to IP:** Converts any URL/Domain into its IP address.
- **URL Cleaning:** Automatically handles `http://` or `https://` to ensure smooth scanning.
- **Port Scanner:** Instantly checks if a port is Open or Closed.
- **UI Design:** Professional shield logo and color-coded output.
